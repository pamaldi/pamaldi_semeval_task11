Convert syllogisms to pyDatalog. Output ONLY executable Python code, no explanations.

# CRITICAL RULES

## 1. File Structure
Every output must be a complete, executable Python file with this structure:
- Imports
- Term declarations
- Facts (with +)
- Rules (with <=)
- Validity check
- Print result

## 2. Proposition Patterns
| Type | Form | pyDatalog |
|------|------|-----------|
| A | "All S are P" | `p(X) <= s(X)` |
| E | "No S is P" | `conflict(X) <= s(X) & p(X)` |
| I | "Some S are P" | `+s('w')` and `+p('w')` |
| O | "Some S are not P" | `+s('w')` (w exists, p('w') NOT asserted) |

## 3. Validity Checks
| Conclusion | Check |
|------------|-------|
| A "All S are P" | `invalid(X) <= s(X) & ~p(X)` then check if empty |
| E "No S is P" | `invalid(X) <= s(X) & p(X)` then check if empty |
| I "Some S are P" | `(s(X) & p(X)).data` check if non-empty |
| O "Some S are not P" | `(s(X) & ~p(X)).data` check if non-empty |

## 4. Valid Forms (memorize)
Fig1: AAA,EAE,AII,EIO | Fig2: EAE,AEE,EIO,AOO | Fig3: IAI,AII,OAO,EIO | Fig4: AEE,IAI,EIO

# EXAMPLES

## Ex1: EIO-1 (Ferio) - VALID
Input: "No soda is a juice. Some beverages are juices. Some beverages are not sodas."

```python
from pyDatalog import pyDatalog
pyDatalog.create_terms('beverage, juice, soda, conflict, X')
+beverage('w')
+juice('w')
conflict(X) <= soda(X) & juice(X)
result = (beverage(X) & ~soda(X)).data
print("VALID" if result else "INVALID")
```
